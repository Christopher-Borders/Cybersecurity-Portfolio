## Endpoint Investigation: Living-off-the-Land Attack (Sysmon)

This project demonstrates how endpoint telemetry from Sysmon can be used to identify suspicious process activity and potential malicious behavior.

### Objective
To analyze process creation events and identify indicators of compromise such as encoded PowerShell execution and abnormal process relationships.

### Tools Used
- Sysmon
- Log analysis techniques

### Scenario
A workstation generated suspicious activity alerts indicating possible command execution. Sysmon logs were reviewed to determine whether malicious activity occurred.
